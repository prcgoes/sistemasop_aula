# 📝 Exercícios

Esta pasta contém os exercícios propostos pelo professor na disciplina **Sistemas Operacionais II**.

## Objetivo

Praticar os conceitos estudados durante as aulas, como:

- gerenciamento de processos
- utilização de comandos do sistema
- programação relacionada ao sistema operacional
- manipulação de arquivos e recursos do sistema

## Organização

Os exercícios são organizados em pastas individuais co numração correspondendo a data:

exercicio05.03  
exercicio06.03  
exercicio03.05

Cada pasta pode conter:

- enunciado do exercício
- código desenvolvido
- arquivos de teste
