# 🔍 Atividade: Práticas com Expressões Regulares (REGEX)

Este pasta contém os exercícios e materiais práticos desenvolvidos para o estudo e aplicação de **Expressões Regulares (REGEX)**.

---

## 📌 Objetivo
O objetivo desta pasta é documentar a capacidade de identificar, validar e extrair padrões de dados complexos em textos não estruturados, tais como:
* 📅 Datas e Horários
* 📞 Contatos Telefônicos e E-mails
* 🏢 Documentos (CNPJ) e Registros (Placas de Veículos)
* 🌐 Links e URLs

---

## 📂 Conteúdo da Pasta
* **`Atividade_REGEX.md`**: Documento completo com o texto base, os comandos utilizados e os resultados obtidos.

---

## 🛠️ Ferramentas Utilizadas
Para a execução e validação das expressões contidas nos documentos, foram utilizadas as seguintes ferramentas:
* [Regex101](https://regex101.com/) - Para testes e depuração de padrões.
* Editor de Texto com suporte a Markdown.
